Primitive Data Types in JavaScript

Primitive data types in JavaScript are the basic building blocks of data. They are immutable (cannot be changed) and are stored by value. Here are the five primitive data types:

1. String

Represents textual data enclosed in single quotes, double quotes, or backticks. 

// Example of a String
let name = "John Doe";
console.log(name); // Output: John Doe

2. Number

Represents numeric data, including integers and floating-point numbers. 

// Example of a Number
let age = 25;
let price = 99.99;
console.log(age);   // Output: 25
console.log(price); // Output: 99.99

3. Boolean

Represents a logical entity with two values: true or false. 

// Example of a Boolean
let isStudent = true;
let isEmployed = false;
console.log(isStudent);  // Output: true
console.log(isEmployed); // Output: false

4. Undefined

Represents a variable that has been declared but not assigned a value. 

// Example of Undefined
let address;
console.log(address); // Output: undefined

5. Null

Represents an intentional absence of any object value. It is explicitly assigned. 

// Example of Null
let salary = null;
console.log(salary); // Output: null

Summary Table

  • String: Textual data. Example: "Hello"
  • Number: Numeric data. Example: 42
  • Boolean: Logical values. Example: true, false
  • Undefined: Unassigned variable. Example: undefined
  • Null: No value. Example: null
