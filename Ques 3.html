Loops in JavaScript

Loops are used to execute a block of code repeatedly as long as a specified condition is true. In JavaScript, loops are essential for automating repetitive tasks and iterating over collections like arrays or objects.

Why Do We Need Loops?

Loops save time and effort by allowing you to write code that runs multiple times without duplicating the same block of code. They are especially useful for:

  • Iterating over arrays or objects.
  • Performing repeated tasks, such as calculations or condition checks.
  • Improving efficiency and reducing the amount of code you need to write.

Types of Loops in JavaScript

1. for Loop

The for loop is the most commonly used loop when you know how many times you want to execute a block of code. It consists of three parts: 

for (initialization; condition; increment/decrement) {
    // Code to be executed
}

Example:

for (let i = 0; i < 5; i++) {
    console.log(i); // Output: 0, 1, 2, 3, 4
}

2. while Loop

The while loop executes a block of code as long as the specified condition is true. It is commonly used when the number of iterations is not known ahead of time. 

while (condition) {
    // Code to be executed
}

Example:

let i = 0;
while (i < 5) {
    console.log(i); // Output: 0, 1, 2, 3, 4
    i++;
}

3. do...while Loop

The do...while loop executes a block of code once, and then repeats the loop as long as the specified condition is true. The key difference from the while loop is that the do...while loop always runs at least once. 

do {
    // Code to be executed
} while (condition);

Example:

let i = 0;
do {
    console.log(i); // Output: 0, 1, 2, 3, 4
    i++;
} while (i < 5);

4. for...in Loop

The for...in loop is used to iterate over the properties of an object. It is useful when you want to loop through the keys or properties of an object. 

for (let key in object) {
    // Code to be executed
}

Example:

let person = {name: "Alice", age: 25, city: "New York"};
for (let key in person) {
    console.log(key + ": " + person[key]); 
}
// Output: name: Alice, age: 25, city: New York

5. for...of Loop

The for...of loop is used to iterate over the values in an iterable object (like an array or string). This is a more modern and efficient way of looping through arrays. 

for (let value of iterable) {
    // Code to be executed
}

Example:

let numbers = [10, 20, 30, 40, 50];
for (let number of numbers) {
    console.log(number); // Output: 10, 20, 30, 40, 50
}
