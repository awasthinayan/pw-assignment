Order list of all HTML Tags

  • For Basic Structure

  1. <html>:Defines the root of an HTML document.
  2. <head>: Contains metadata about the document (title, links, scripts, etc.).
  3. <title>:Sets the title of the document (displayed in the browser tab).
  4. <body>:Contains the visible content of the web page.

  • For Text Content

  1. <h1> to <h6>: Headings (h1 is the largest, h6 is the smallest).
  2. <p>: Paragraph.
  3. <br>: Line break.
  4. <hr>: Horizontal rule (divider).
  5. <pre>: Preformatted text (preserves spaces and line breaks).
  6. <b> / <strong>: Bold text (strong adds emphasis).
  7. <i> / <em>: Italic text (em adds emphasis).
  8. <u>: Underlined text.
  9. <small>: Smaller text.
  10. <mark>: Highlighted text.
  11. <blockquote>: Quoted text (usually indented).
  12. <q>: Inline quotation.
  13. <abbr>: Abbreviation (provides a tooltip).
  14. <cite>: Cites a source or reference.
  15. <code>: Inline code.
  16. <sup>: Superscript text.
  17. <sub>: Subscript text.

  • For Links and Navigation

  1. <a>: Hyperlink.
  2. <nav>: Defines navigation links.

  • For Lists

  1. <ul>: Unordered list (bullets).
  2. <ol>: Ordered list (numbers).
  3. <li>: List item (used inside <ul> or <ol>).
  4. <dl>: Definition list.
  5. <dt>: Definition term.
  6. <dd>: Definition description.

  • For Images and Multimedia

  1. <img>: Image.
  2. <figure>: Groups content like images and captions.
  3. <figcaption>: Caption for <figure>.
  4. <audio>: Audio content.
  5. <video>: Video content.
  6. <source>: Specifies multiple media resources for <audio> or <video>.
  7. <embed>: Embeds external content (like videos).
  8. <iframe>: Inline frame for embedding external web pages.

For Tables

  1. <table>: Table.
  2. <tr>: Table row.
  3. <td>: Table data cell.
  4. <th>: Table header cell.
  5. <thead>: Groups the table header.
  6. <tbody>: Groups the table body.
  7. <tfoot>: Groups the table footer.
  8. <caption>: Table caption.
  9. <col>: Specifies column properties.
  10. <colgroup>: Groups columns.

  • For Forms and Input

  1. <form>: Form container.
  2. <input>: Input field (text, password, etc.).
  3. <textarea>: Multi-line text input.
  4. <button>: Button.
  5. <label>: Label for form elements.
  6. <select>: Dropdown menu.
  7. <option>: Options within <select>.
  8. <fieldset>: Groups related form elements.
  9. <legend>: Title for <fieldset>.
  10. <datalist>: Predefined list of input values.
  11. <output>: Outputs a calculation or result.
  • h2>For Styling and Semantics
  1. <div>: Generic container (block-level).
  2. <span>: Generic inline container.
  3. <header>: Header section.
  4. <footer>: Footer section.
  5. <main>: Main content area.
  6. <article>: Self-contained content.
  7. <section>: Thematic grouping of content.
  8. <aside>: Sidebar or related content.
  9. <details>: Disclosure widget.
  10. <summary>: Summary for <details>.

  • For Scripts and Links

  1. <script>: Embeds or references JavaScript.
  2. <link>: Links external resources (e.g., CSS files).
  3. <style>: Embeds internal CSS styles.
  4. <meta>: Metadata about the HTML document.

  • For Other tags

  1. <canvas>: Drawing area for graphics (requires JavaScript).
  2. <svg>: Scalable vector graphics.
  3. <noscript>: Content for users without JavaScript enabled.
  4. <progress>: Progress bar.
  5. <meter>: Measurement or gauge.
