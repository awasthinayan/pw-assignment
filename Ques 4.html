Declaring Variables in JavaScript

Purpose of Declaring Variables

Variables are containers for storing data values. In JavaScript, declaring variables allows developers to:

  • Store and reuse data.
  • Make the code more readable and maintainable.
  • Perform operations on stored values.

Using the let Keyword

The let keyword is used to declare variables in JavaScript. Variables declared with let are:

  • Block-scoped: They are accessible only within the block, statement, or expression they are defined in.
  • Reassignable: Their values can be changed after declaration.

Syntax

let variableName = value;

Examples

1. Declaring and Assigning a Variable

// Example: Declaring and assigning a variable
let name = "Alice";
console.log(name); // Output: Alice

2. Changing the Value of a Variable

// Example: Changing a variable's value
let age = 25;
console.log(age); // Output: 25
age = 30; // Reassigning a new value
console.log(age); // Output: 30

3. Block Scope Behavior

// Example: Block scope with 'let'
if (true) {
    let color = "blue";
    console.log(color); // Output: blue (accessible inside the block)
}
console.log(color); // Error: color is not defined (outside the block)
